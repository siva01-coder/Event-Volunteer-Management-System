# Event Volunteer Management System

## Overview

The Event Volunteer Management System is a web-based platform developed using Django and MongoDB to simplify volunteer registration, event management, task allocation, and participation tracking. The system helps organizers efficiently coordinate volunteers and manage events through a centralized dashboard.

## Features

* User Registration and Login
* Volunteer Dashboard
* Event Creation and Management
* Volunteer Registration for Events
* Task Assignment and Tracking
* Participation Monitoring
* Admin Dashboard
* MongoDB Database Integration

## Technologies Used

* Django
* MongoDB
* HTML
* CSS
* JavaScript

## Project Structure

* Accounts Management
* Event Management
* Volunteer Management
* Task Allocation
* Database Integration

## Installation

1. Clone the repository
2. Install dependencies using:
   pip install -r requirements.txt
3. Configure MongoDB connection
4. Run the server:
   python manage.py runserver
